üåæUSDA FoodData Central Integration

  • Smart Food Matching: Advanced fuzzy matching algorithm finds the best food matches
  • Priority System: Prioritizes Foundation foods > SR Legacy > Survey (FNDDS) > Branded
  • Comprehensive Search: Searches across multiple USDA food databases
  • Nutrition Accuracy: Uses official USDA nutritional data
  • Fallback Handling: Graceful handling of missing or incomplete data
  • Performance Optimized: Caching and efficient API usage

Matching Algorithm:

1. Exact match (case-insensitive) 2. Description starts with query 3. Description contains full query 4. Comprehensive scoring: - Word matching score - Data type priority boost - Calorie availability boost - Description length penalty
