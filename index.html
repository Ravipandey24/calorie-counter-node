🌱USDA FoodData Central Integration

  • Smart Food Matching: Multi-tier fuzzy matching algorithm with intelligent scoring
  • Priority System: Foundation > SR Legacy > Survey (FNDDS) > Branded foods
  • Comprehensive Search: POST-based search across all USDA food databases
  • Nutrition Accuracy: Official USDA nutritional data with macronutrient breakdown
  • Fallback Handling: Graceful handling of missing data with energy unit conversion
  • Performance Optimized: 10-second timeout with sorted results by data type

4-Tier Matching Algorithm:

1. Exact match (case-insensitive description matching) 2. Starts with query (description begins with search term) 3. Contains full query (description includes complete search term) 4. Comprehensive scoring: ├── Word matching score (query words in description) ├── Data type priority boost (+10 to +20 points) ├── Calorie availability boost (+10 points) └── Description length penalty (-5 for long descriptions) Energy Conversion: Automatic kJ to kcal conversion (1 kcal = 4.184 kJ) Macronutrients: Protein, fats, carbs, fiber, sugars per 100g
